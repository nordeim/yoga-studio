✦ The LuxeVerse Architect Skill is a comprehensive architectural and execution framework designed to build cinematic, luxury-grade web platforms. It moves beyond generic "AI-slop" or template-based development, focusing on high-end
design, technical rigor, and a phased delivery system.

Core Purpose
Its primary goal is to guide the development of high-performance, accessible, and visually unique applications (specifically targeting "Anti-Generic" luxury standards) using a cutting-edge tech stack.

Key Pillars
 * Anti-Generic Mandate: Rejects common design tropes (like standard bento grids or stock hero sections) in favor of bespoke, asymmetric layouts and intentional whitespace.
 * Performance as Luxury: Enforces strict metrics (LCP < 2.5s, CLS < 0.1) and compositor-only animations to ensure "white-glove" responsiveness.
 * Accessibility as Law: Mandates WCAG AA/AAA compliance, including focus traps, skip links, and motion reduction support.
 * Modern Tech Stack:
     * Next.js 16 (App Router & RSC)
     * React 19 (useActionState, useOptimistic)
     * TypeScript 6 (Strict mode, no any)
     * Tailwind CSS 4 (CSS-first configuration)
     * Zustand 5 (Flat state management)
     * Zod 4 (Runtime validation)

6-Phase Execution Framework
The skill organizes work into sequential, verifiable stages:
 1. Foundation: Design system, tokens, and CI/CD setup.
 2. Core functionality: Business logic (e.g., Commerce, Auth, API).
 3. Cinematic Experience: Motion graphics, 3D elements, and editorial layouts.
 4. AI & Personalization: Intelligent features and streaming responses.
 5. Scale & Loyalty: i18n, PWA, and social features.
 6. Polish & Launch: TDD, E2E testing, and production hardening.

Agent Workflow
When this skill is active, I follow a strict ANALYZE → PLAN → VALIDATE → IMPLEMENT → VERIFY → DELIVER workflow. This ensures that no code is written without a validated plan and every deliverable passes rigorous quality gates
(Lighthouse, security audits, and type-checking).

Essentially, it is a master blueprint for delivering production-ready, architecturally sound, and visually striking web experiences.
