📐 Architecture 
 
5-Layer Golden Rule: 
 
``` 
  Layer 0: src/proxy.ts          → Cookie check, redirect (Edge runtime) 
  Layer 1: src/app/              → Routes, metadata, Suspense 
  Layer 2: src/features/         → UI composition, data binding, mutations 
  Layer 3: src/features/*/domain/ → Pure business logic 
  Layer 4: src/lib/              → Infrastructure (DB, Auth, AI, Storage, Stripe) 
```
